SynScan 3.9b2 Released, with IPv6 Support

Version 3.9b2 of SynScan is an early beta of the forthcoming SynScan 4.0. As far as I'm aware, this is the first half open syn-scanner with support for IPv6.

Currently there is support for IPv6 scanning, but with several limitations:
There is no synscand (you can only identify open ports, there are no vuln-checks or banner grab support)
There is no support for IPv4 again yet, this beta concentrated on getting IPv6 working
You can only scan from an IP List, SynScan cannot generate a list of sequential IPv6 addresses yet (and such lists would be huge, anyway).

SynScan 3.9b2 as well as earlier versions are available from:
http://www.bindshell.net/tools/synscan
